# 🚨 Crime Forecasting & Hotspot Visualization Dashboard

An interactive, data-driven web application built with **Streamlit** to analyze, forecast, and visualize crime patterns across India. 

---

## ✨ Key Features

* 🔄 **Automated Data Ingestion:** Automatically downloads and processes the latest NCRB (National Crime Records Bureau) data directly from the Government of India portal.
* 📊 **State-Level Analytics:** Granular crime data forecasting and detailed mapping categorized by individual states.
* 🗺️ **Live Risk Mapping:** Dynamic, interactive risk map overlays utilizing precise state coordinates.
* 🚀 **Cloud-Deployment Ready:** Fully configured for seamless Streamlit deployment with native support via `runtime.txt`.

---

## 🛠️ Tech Stack

* **Frontend & Dashboard:** Streamlit
* **Core Language:** Python
* **Data Processing:** Pandas, NumPy
* **Machine Learning / Forecasting:** Scikit-Learn, XGBoost, LSTM
* **Visualization:** Streamlit Native Charts, Matplotlib, Seaborn
